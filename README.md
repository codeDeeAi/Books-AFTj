
## About Books

Books is a simple web application built with vue js and laravel. Users can Signup, create books and read books created by other users

## About  API's

    ## api/book
        creates a book

    ## api/books
        returns all books 

    ## api/book/{id}
        returns a book 

    ## api/book/{id}
        deletes a book
